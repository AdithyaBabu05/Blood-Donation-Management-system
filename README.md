Project Overview
The Blood Donation Management System is a web-based application designed to simplify and organize the process of blood donation and blood stock management.
It helps manage donor information, blood stock, donation records, and blood requests efficiently through a centralized system.
This project was developed as part of an academic mini project to understand full-stack web development, database integration, and basic system design.

Objectives
1)Maintain donor records in an organized manner
2)Track available blood stock by blood group
3)Record blood donations and requests
4)Enable easy searching of eligible donors
5)Reduce manual effort and improve data accuracy

Technologies Used
Frontend: HTML, CSS
Backend: Python (Flask)
Database: SQLite
Tools: VS Code, Git, GitHub
